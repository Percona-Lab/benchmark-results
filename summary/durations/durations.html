Setup

  • Client and server on the same machine
  • Sysbench oltp workload running during all tests
  • 21GB datadir (restored between tests)
  • CPU: 48 logical CPU threads (Intel® Xeon® CPU E5-2680 v3 @ 2.50GHz).
  • Memory: 120GB.
  • Disk: hdd (HGST HUH728080ALE600), ssd (Intel 3600 nvme)
  • Encryption (E) and Compression © tested with 1, 4, 8, 16 and 32 threads
